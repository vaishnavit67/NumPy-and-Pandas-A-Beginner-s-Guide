# Data Science Fundamentals: NumPy, Pandas & Matplotlib
## A CheatSheet for Data Science Fundamentals

Heyyy! This repository contains a collection of hands-on reference guides and practiced notebooks for the three core pillars of Python data science: **NumPy**, **Pandas**, and **Matplotlib**.

It is designed to serve as a clean, simple code reference for anyone looking to learn, practice, or quickly copy syntax for data manipulation and visualization.
